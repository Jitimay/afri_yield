## 🎯 **Design System**

### **Color Palette**
- **Primary**: Emerald (farmers) → `from-emerald-500 to-green-500`
- **Secondary**: Blue (lenders) → `from-blue-500 to-indigo-500`
- **Accent**: Purple (transparency) → `from-purple-500 to-pink-500`
- **Success**: Green → `from-green-500 to-emerald-500`
- **Warning**: Orange → `from-orange-500 to-red-500`

### **Components**
- **Cards**: `rounded-2xl shadow-lg border border-gray-100`
- **Buttons**: Gradient backgrounds with hover animations
- **Inputs**: `rounded-xl` with focus states
- **Headers**: Gradient backgrounds with icons

### **Animations**
- **Hover effects**: `transform hover:scale-105`
- **Card animations**: `hover:-translate-y-2`
- **Loading spinners**: Custom CSS animations
- **Smooth transitions**: `transition-all duration-200`
